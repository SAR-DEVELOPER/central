## Features

When users access their specific KAP link:
- ✅ The **Nama Kantor Akuntan Publik (KAP)** field is automatically pre-filled with the KAP name
- ✅ The KAP field is **disabled** to prevent changes
- ✅ The **Nama Klien KAP** dropdown shows only the companies assigned to that specific KAP
- ✅ Users can search and select from their assigned client list

## Usage

Share the appropriate link with each KAP. Each KAP will only see and be able to select from their assigned client companies.
